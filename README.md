# Modern Scientific Calculator

A beautifully designed, responsive, and fully functional Scientific Calculator built with HTML, CSS, and Vanilla JavaScript. It features a modern dark-themed UI with glassmorphism aesthetics, inspired by the Material Design 3 and Windows Calculator interfaces.

## 🌟 Features

- **Scientific Functions**: Includes square (`x²`), square root (`²√x`), reciprocals (`¹/x`), absolute values (`|x|`), exponents (`exp`), modulus (`mod`), factorials (`n!`), logarithms (`log`, `ln`), powers of 10 (`10ˣ`), and standard `xʸ`.
- **Beautiful UI**: Distinctly colored buttons for numbers, operators, and functions to enhance visual clarity and usability. A vibrant equalizer button stands out for quick access.
- **History Drawer**: A hidden history panel that smoothly slides out, allowing you to review past calculations and reload previous results.
- **Keyboard Support**: Full keyboard mapping for numbers, operators, `Enter` (equals), `Backspace`, and `Escape` (clear).
- **Sound Effects**: Built-in, zero-dependency sound effects using the Web Audio API for a satisfying click experience.
- **Responsive Layout**: Adapts perfectly to mobile screens with touch-friendly button sizes.

## 📥 Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/sailesh-01/Simple-Calculator.git
   ```
2. Navigate to the directory:
   ```bash
   cd Simple-Calculator
   ```
3. Open `index.html` in your favorite web browser. No build steps or servers required!

## 💡 Usage

- **Click or Type**: You can click the on-screen buttons or type directly using your keyboard.
- **History**: Click the history icon (clock arrow) at the top right to open the history drawer. Click any past calculation to load it back onto the screen.
